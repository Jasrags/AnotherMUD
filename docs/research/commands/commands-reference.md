# Aardwolf — Command Reference (by Category)

Source: https://wiki.aardwolf.com/index.php/Help/Contents + category pages
Fetched: 2026-06-09

This document contains the full categorized command/help listing from AardWiki.
Each entry is: `Command : Description`

---

## Help System Categories

```
Area         : Non-goal information about areas on Aardwolf.
Clans        : Clan information, policies, and related information.
Classes      : Detailed information on each class and subclass.
Communication: Ways to interact with others on Aardwolf.
Config       : Settings found in the 'config all' command.
Features     : Explains Aardwolf's main features in more detail.
Goal         : Information on areas that have goals added.
Information  : MUD information not suited for other categories.
Miscellaneous: Skills/spells/information not found in other categories.
Newhelp      : Information tailored for newbies.
Races        : Detailed information on each race's unique features.
Rules        : MUD policies and rule-specific help files.
Utility      : Commands that configure your character / MUD reactions.
```

Skills/Spells:
```
Attack Skill : Forms of physical combat not involving magic.
Attack Spell : Magical energies of a destructive nature.
Detection    : Means to see the otherwise undetectable.
Enhancement  : Various temporary ways to buff up yourself and others.
Equipment    : Creation and modification of in-game objects.
Followers    : List of different pets and how to interact with them.
Healing      : Various abilities to return to full health.
Malediction  : Various ways to deteriorate opponents' combat abilities.
Protection   : Spells/abilities that help with defense during combat.
Survival     : Tricks and tools used to survive longer in the wild.
Transport    : Aids for quickening travel around the realm.
```

Immortal-Only:
```
Admin        : Immortal commands, utilities, and information.
OLC          : Imm-only building helpfiles.
```

  https://www.aardwolf.com/wiki/index.php/Help/[CommandName]

- Commands listed as "toggle" commands can typically be typed again to reverse the effect
- Skill/spell availability varies by class and level
