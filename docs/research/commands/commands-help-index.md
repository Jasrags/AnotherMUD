## Notable Command Categories (from section structure)

### Movement & Navigation
- Standard cardinal directions: n, s, e, w, ne, nw, se, sw, u, d
- MOVEMENT — overview of movement
- LANDMARKS — auto-move to known locations
- PORTALS — enter magical portals
- RECALL — return to safety
- EXITS — see available exits

### Inventory & Items
- INVENTORY / INV — see what you're carrying
- GET / TAKE — pick up items
- DROP — drop items
- PUT — put items into containers
- WEAR / WIELD / REMOVE — equip/unequip items
- LOOK AT — examine specific items
- CONTAINERS / PACKS — container management

### Communication (Section 3)
Achaea has a full communication section. Key commands include:
- SAY / ' — speak to everyone in the room
- TELL — private message to a specific player
- SHOUT — heard across a large area
- YELL — heard in adjacent rooms
- EMOTE / ; — perform an action
- CHANNELS — various global channels (CT for city-talk, etc.)
- WHO — see who is online

### Character (Section 4)
- SCORE / SC — see your character's stats
- SKILLS — list your skills
- AFFECTS — see active buffs/debuffs on you
- ACHIEVEMENTS — track accomplishments
- DESCRIPTION — set your character description
- TITLE — set your character title

### Combat (Section 13)
- ATTACK / KILL — initiate combat
- FLEE — attempt to escape combat
- CONSIDER — gauge difficulty of a target
- GROUP — manage combat groups/parties

### Housing (Section 20)
- BUILD — add rooms to your house
- DESCRIBE ROOM — set room description
- FURNISH — place furniture
- DOOR — manage doors
- EXIT — manage room exits
- SERVANTS — manage hired servants

### Ships & Seafaring (Section 23)
- SAIL — sail your ship
- NAVIGATE — navigate to a destination
- BOARD — board a ship
- DISEMBARK — leave a ship
- CREW — manage ship crew
- FIRE — fire ship weapons

### Mines & Legions (Section 24)
- MINE — mine-related commands
- LEGION — manage your legions
- PROSPECT — look for mineral lodes
- DEPLOY — deploy legions to a location
- ATTACK (LEGION) — send legions to assault

## Notes on Achaea's Help System

- All help files accessible in-game: HELP <topic> or HELP <section number>
- Online mirror: https://www.achaea.com/game-help/?what=<topic-slug>
- Each section has numbered subsections (e.g., HELP 2.9 for basecommands)
- The system is extensive — 24 top-level sections with multiple subsections each
- Individual skill/ability help files exist for all class skills

## Key Differences from Aardwolf

- Achaea help is section-numbered (2.4.1) vs Aardwolf's keyword-based (HELP MOVEMENT)
- Achaea has dedicated sections for Ships (23) and Mines (24) — unique content areas
- Achaea exposes all help online at a URL-based system
- No global "COMMANDS" command listed — discovery through HELP sections
